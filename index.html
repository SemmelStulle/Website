This is a list of features this site supports:

- switches light mode with system/browser
- favicons for almost all use cases
- can be a PWA on iOS (add to home screen and it becomes a window with no decorations)
- blur for the footer
- an easter egg
- separated CSS and JS for better readability
- comments in the code
- tries to use the OS font to make it feel more native
- uses as little JavaScript as possible
- as compatible with OS features and accessibility as I can get it to be

This website is optimized for WebKit and WebRender. Please visit from Safari or Firefox for best results.
