Hier eine Liste der Features, die diese Website unterstützt:

- Wechselt zwischen hellem und dunklem Modus mit dem Browser/dem OS
- Favicons für fast alle Zwecke
- Kann als PWA installiert werden (ein Fenster ohne Browser Dekoration)
- Unschärfe für den footer
- Ein Easter Egg
- Separates CSS und JS für bessere Lesbarkeit
- Kommentare im Code
- Benutzt die Schrift des OS, wenn bereitgestellt
- Verwendet möglichst wenig JavaScript
- Benutzt möglichst viele OS Features für eine native Benutzung
- Benutzt LocalStorage
- Ist mehrsprachig (aktuell Englisch und Deutsch, Spanisch kommt eventuell ebenfalls)

Diese Website ist für WebKit und WebRender optimiert. Für die Beste Erfahrung, besuchen Sie die Seite bitte mit Safari oder Firefox.
